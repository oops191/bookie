## 图片

插入图片的语法和链接类似，前面多一个感叹号 `!`：

```markdown
![图片描述文字](图片的URL地址)
```

具体示例：

```markdown
![一张风景照](https://example.com/photo.jpg)
```

三者含义：

- `!`：表示这是一张图片
- `[图片描述文字]`：图片的替代文本（alt），在图片无法加载时显示，也给搜索引擎和视障读者提供信息
- `(URL)`：图片的链接地址

URL 可以来自两个地方：

1. **网络图片**：直接粘贴图片网址
2. **本地图片**：把图片放到 `posts/` 目录下，用相对路径引用

```markdown
![我的照片](my-photo.jpg)
```

## 如何新建一篇文章

总共三步，非常简单。

### 第一步：创建 .md 文件

在 `posts/` 目录下新建一个 `.md` 文件，文件名就是文章的 slug（URL 标识），建议用英文加短横线：

```
posts/my-first-article.md
```

### 第二步：写文章

文件第一行必须是日期注释，第二行是标题。示例：

```markdown
<!-- date: 2026-06-01 -->

# 我的第一篇文章

这是正文内容。段落之间用空行分隔。

## 小标题

正文继续……
```

**必须遵守的规则：**

- 第一行：`<!-- date: YYYY-MM-DD -->`
- 第二行：`# 文章标题`
- 之后自由写正文，使用上述各种语法即可
- `<!-- date: -->` 和 `# 标题` 不要在正文里重复出现

### 第三步：推送

写完保存，推送到 GitHub 即可。GitHub Action 会自动扫描新文章并更新索引，无需手动操作。

```bash
git add .
git commit -m "新文章：我的第一篇文章"
git push
```

如果想要在本地预览效果，推送前先运行：

```bash
python generate.py
python -m http.server 8080
```

然后浏览器打开 `http://localhost:8080` 查看。
